multer (^1.4.4)
multer is a middleware for handling multipart/form-data, primarily used for file uploads.
It allows you to store uploaded files in memory or on disk before passing them to the request handler.

multer-gridfs-storage (^5.0.2)
This package extends multer to store files directly in MongoDB GridFS instead of the file system.
GridFS is used when files are too large to fit in a MongoDB document.
It enables efficient retrieval and streaming of large files.
